# 《塔陨：异步终局》Playwright专业复测与P0修复（2026-03-19）

## 1. 复测方式
- 使用 `user-Playwright` MCP 在 `http://127.0.0.1:8080` 实机跑流程
- 路径：开局 -> 购买单位/道具 -> 探索 -> 手动封盘 -> 异步PVP -> 下一层
- 重点检查：路径选择有效性、战斗可视化、关键反馈清晰度

## 2. 核心发现（实测结论）
- **P0-1：路径选择与行动执行错配**
  - 现象：选择了 `shop` 节点后点击“探索”，仍触发战斗并扣血（日志出现 `节点[shop] 探索战斗 ...`）。
  - 影响：杀戮尖塔式分支决策失真，玩家学习成本和挫败显著增加。
- **P0-2：关键日志重复感知**
  - 现象：日志区“最新关键结论”与正文首条重复，视觉噪音高。
  - 影响：复盘效率下降，关键信息不聚焦。
- **P1：战斗可视化已有基础但仍可深化**
  - 已有：浮窗、回放轨道、暂停/倍速、双阵营单位与血条。
  - 待加强：目标选择规则说明、单位技能触发特效、可复盘关键帧跳转。

## 3. 已完成修复（本次落地）
- **修复 P0-1（已落代码）**
  - 路径节点强约束动作：
    - `combat/elite/boss` 仅允许“探索”
    - `event/shop/rest` 仅允许“触发事件”
  - 错误动作会提示，不再消耗节点。
  - 按钮可用状态会随当前选中节点自动变化。
- **修复 P0-2（已落代码）**
  - 日志关键高亮与正文去重，避免首条重复显示。

## 4. 当前状态评估（专业角度）
- 五大玩法已可感知，且“战斗过程浮窗”已从静态文本升级为动态回放。
- 目前最关键风险已从“玩法缺失”转为“数值与反馈打磨”阶段。
- 下一优先应聚焦：
  1. 战斗结果可解释性（克制链、站位价值）  
  2. 后期数值压迫（终局惩罚与经济上限）  
  3. 路径分支奖励差异化（让选路真的有风格）

## 5. 下一步建议（可直接开发）
- 增加“节点执行预览卡”：执行前显示预估收益/风险区间
- 增加“战斗关键帧跳转”：开场、首减员、胜负拐点一键跳转
- 增加“路径派系奖励”：连续选择同类节点触发额外增益
