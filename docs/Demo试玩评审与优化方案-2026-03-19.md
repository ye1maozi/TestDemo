# 《塔陨：异步终局》Demo试玩评审与优化方案（2026-03-19）

## 1. 评审范围与方法
- 试玩环境：本地 `http://127.0.0.1:8080` Demo（Playwright MCP）
- 试玩流程：完成开局到第3层，覆盖引导、探索/事件、商店购买、背包整理、封盘、异步PVP结算
- 评审视角：以“策略深度可读性、节奏压迫感、成长反馈、长期可运营性”为核心

## 2. 总体结论（策划视角）
- **方向正确**：异步快照对战 + 爬塔循环的主框架完整，已具备“可反复跑一局”的雏形。
- **当前短板集中在中前期体验**：玩家对“我为什么赢/输、下一步该做什么”理解成本偏高。
- **高优先级应先做可读性与阶段目标**：先解决认知与反馈，再扩内容池，收益最大。

## 3. 试玩关键观察（证据）
- 第1~3层可稳定进入：`自由探索 -> 路径收缩 -> 封盘提交 -> 异步匹配结算 -> 下一层`
- 日志可见核心链路（示例）：
  - `封盘完成，正在请求异步PVP结算...`
  - `PVP失败(mock) 对手=镜像对手-xxxx HP-xx`
  - `楼层结算：金币+x，恢复xHP，预计剩余xx人`
- 新手引导可跳过，但“操作目标->收益变化->战力变化”之间的映射仍不直观。

## 4. 问题与优化方案（按优先级）

### P0：战斗结果可解释性不足（影响留存）
**问题**
- 结算时仅看到“输赢+扣血”，难理解输因（站位、阵容、装备触发、节奏失误）。

**优化方案**
- 增加“失败归因卡”（结算页前置）：
  - 我方伤害占比前三单位
  - 关键被克制点（前排承压不足/后排被切/控制覆盖不足）
  - 三条可执行改进建议（如“前排+1坦度”“补单体爆发”“背包补护盾触发”）
- 增加“快照对战事件轴”最小版：
  - 开场5秒关键触发、首个减员时刻、胜负拐点

**验收指标**
- 新手局后问卷“我知道为什么输”同意率 >= 65%
- 第1局~第3局中途退出率下降 >= 15%

---

### P0：自由探索阶段决策目标不清晰（影响策略乐趣）
**问题**
- 探索/事件/购物/布阵在同一窗口并行，但玩家缺“本层最优目标提示”。

**优化方案**
- 每层开始给出“本层任务导向”二选一：
  - 稳健线：保血保徽记（推荐操作优先级）
  - 贪收益线：冲经济冲上限（承受更高失败风险）
- HUD新增“本层建议动作序列”：
  - 例如：`先买前排 -> 再事件 -> 最后整理背包`
- 对关键按钮增加收益预估浮层（点击前可见）：
  - `探索节点：预计 +2~4 金币，受伤风险中`
  - `触发事件：预计收益波动高`

**验收指标**
- 首局玩家“无操作停顿超过8秒”的次数下降 >= 20%
- 第1层自由探索阶段有效操作次数提升 >= 25%

---

### P1：背包与棋盘联动感弱（影响核心卖点）
**问题**
- 已有背包整理与棋盘站位，但玩家难感知“背包布局如何影响战斗结果”。

**优化方案**
- 在背包整理后即时显示“战前加成摘要”：
  - 护盾 +x%、暴击 +x%、技能冷却 -x%
- 在结算页显示“本局已触发背包词条次数Top3”
- 增加“推荐摆放高亮”（仅建议，不代替策略）：
  - 当存在可触发邻接加成时给出弱提示框

**验收指标**
- 玩家平均每层主动整理背包次数 >= 1.2
- 背包相关提示点击率 >= 40%

---

### P1：时间压力节奏偏硬，缺恢复机制（影响挫败感）
**问题**
- 路径收缩与封盘节奏明确，但连续失利时容易进入“认知崩盘 -> 连败”。

**优化方案**
- 引入“战术缓冲券”（每局1次）：
  - 可将封盘延后5秒，仅限前3层使用
- 连败保护强化（前3层）：
  - 连败2次触发一次低风险补给事件（小额保底）
- 第2层结算增加“系统建议卡”（自动生成一条最优补强）

**验收指标**
- 前3层徽记归零淘汰率下降 >= 20%
- 首日对局完成率提升 >= 8%

---

### P2：运营层“复仇/宿敌”表达不足（影响异步对抗张力）
**问题**
- 异步对战虽然稳定，但“人对人情绪”偏弱，吃鸡紧张感可再加强。

**优化方案**
- 新增“宿敌标记”：
  - 同一对局中重复交手2次后形成宿敌关系，结算演出强化
- 新增“复仇奖励”：
  - 击败曾战胜你的对手，获得少量额外秘能/荣誉点
- 终局前展示“潜在冠军竞争者卡片”增强压迫感

**验收指标**
- 对局末段（7层后）活跃操作频率提升 >= 10%
- 终局阶段停留时长提升 >= 12%

## 5. 版本落地建议（两周冲刺）
- **Sprint A（优先）**
  - 失败归因卡（P0）
  - 本层任务导向与操作建议（P0）
  - 战前加成摘要（P1）
- **Sprint B（增强）**
  - 战术缓冲券 + 连败保护优化（P1）
  - 宿敌/复仇系统最小版（P2）

- 在 `异步爬塔吃鸡GDD-v2.md` 增补“结果可解释性”为首要体验目标（与高策略定位一致）

- `loss_reason_panel_view_rate`（失败归因卡查看率）
- `backpack_bonus_summary_click_rate`（背包加成摘要点击率）
- `first3floor_elimination_rate`（前3层淘汰率）
